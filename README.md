# heart-disease-prediction
Best Algorithm: K-Nearest Neighbors (KNN)

Based on the obtained results, the K-Nearest Neighbors (KNN) algorithm achieved the best performance in predicting heart disease. Compared to the other models, KNN obtained higher Accuracy, Precision, and Recall values. Therefore, it was selected as the best model for this dataset.

Since the Accuracy, Precision, and Recall values for KNN were all approximately 0.61 (61%), and these values were higher than those of the Decision Tree (DT) and Naive Bayes (NB) models, this conclusion is consistent with the results of the project.
